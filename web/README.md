
# Ibn Al‑Nasr Battle (Starter Project)

This is a **starter structure** for a Unity battle‑royale style project requested by the user.
It includes placeholder scripts and documentation so a Unity developer can open the project
and continue building the game.

Main design goals:
- Iraq map (lightweight)
- Detailed Nasiriyah area
- 50 real players (online)
- Voice chat
- Login system (Google / Facebook / Phone via Firebase)
- Plane drop + parachute
- Vehicles
- Weapons
- Custom rooms

Important:
3D models such as:
- Imam Husayn shrine
- Imam Ali shrine
- Nasiriyah city assets

must be imported from external 3D assets or created in Blender.

Open with Unity Hub and create a new project, then copy the Assets folder into it.
